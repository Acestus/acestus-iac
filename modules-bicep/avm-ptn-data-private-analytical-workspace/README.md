# Private Analytical Workspace Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/data/private-analytical-workspace` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module privateAnalyticalWorkspace 'modules-bicep/avm/ptn/data/private-analytical-workspace/main.bicep' = {
  name: 'privateAnalyticalWorkspace'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.1.2`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
