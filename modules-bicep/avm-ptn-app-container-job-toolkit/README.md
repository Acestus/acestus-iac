# Container Job Toolkit Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/app/container-job-toolkit` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module containerJobToolkit 'modules-bicep/avm/ptn/app/container-job-toolkit/main.bicep' = {
  name: 'containerJobToolkit'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.1.3`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
