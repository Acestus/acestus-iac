# AI Platform Baseline Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/ai-platform/baseline` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module aiPlatformBaseline 'modules-bicep/avm/ptn/ai-platform/baseline/main.bicep' = {
  name: 'aiPlatformBaseline'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.8.0`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
