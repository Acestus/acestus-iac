# AI ML Foundry Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/ai-ml/ai-foundry` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module aiMlFoundry 'modules-bicep/avm/ptn/ai-ml/ai-foundry/main.bicep' = {
  name: 'aiMlFoundry'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.6.0`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
