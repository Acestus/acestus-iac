# ALZ Empty Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/alz/empty` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module empty 'modules-bicep/avm/ptn/alz/empty/main.bicep' = {
  name: 'empty'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.3.6`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
