# ALZ AMA Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/alz/ama` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module ama 'modules-bicep/avm/ptn/alz/ama/main.bicep' = {
  name: 'ama'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.1.1`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
