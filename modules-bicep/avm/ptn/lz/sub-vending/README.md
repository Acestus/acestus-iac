# Sub Vending Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/lz/sub-vending` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module subVending 'modules-bicep/avm/ptn/lz/sub-vending/main.bicep' = {
  name: 'subVending'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.5.3`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
