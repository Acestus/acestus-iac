# Modernize Your Code Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/sa/modernize-your-code` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module modernizeYourCode 'modules-bicep/avm/ptn/sa/modernize-your-code/main.bicep' = {
  name: 'modernizeYourCode'
  params: {
    // Add parameters here
  }
}
```

- Uses AVM module version: `0.2.0`
- Opinionated wrapper for Acestus standards
