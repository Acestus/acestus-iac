# Chat With Your Data Wrapper Module

## Overview
This module wraps the AVM module `avm/ptn/sa/chat-with-your-data` for Acestus deployments. Version: **1.0.0**

## Usage
```bicep
module chatWithYourData 'modules-bicep/avm/ptn/sa/chat-with-your-data/main.bicep' = {
  name: 'chatWithYourData'
  params: {
    // Add parameters here
  }
}
```

## Implementation
- Uses AVM module version: `0.2.0`
- Opinionated wrapper for Acestus standards

## Security
- Follows Acestus security best practices
